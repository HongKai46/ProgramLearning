# 运算符

11/5=2.2

11//5=2

3**3=27

字符串类型与数字类型不能相运算（1+“3”不行） 但可以先类型转换

# 字符串快速格式化

```python
a=66
b=f"hello {a} world"
print(b)
print(f"hello {a} world")
```

# 条件语句if

```python
a=21

	print("adult")
elif a<18:
	print("child")
else:
    print("18")
```
