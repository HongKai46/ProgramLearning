# 列表[list]

## 访问元素

`a[-1]` 表示访问倒数第一个元素

## .append()

**功能：从末尾追加一个元素**

参数为**一个**任意类型元素

## .extend()

**功能：从末尾追加一个数据容器内的所有元素**

参数为一个数据容器（列表，字符串，etc）

## .index()

**功能：查找指定元素的下标，元素不存在，会报错**

参数可以传入三个：

1.寻找的元素值

2.查找的起始下标(默认为起始）

3.查找的终止下标(默认为最后)

## .count()

**功能：查询指定元素的数量**

参数即为一个要查询的元素

## .insert()

**功能：在指定位置插入一个元素**

## .pop()

**功能：删除指定下标的元素**

参数有一个（必要）：指定下标

## .remove()

**功能：删除第一个指定的元素，如果元素不存在会报错**

参数为一个：元素本身

参数有两个（必要）：1.位置下标 2.需要插入的元素

## .clear()

**清空列表，无参数**

```python
a=[1,2,3,"hello"]
a.append(2) #添加一个元素
#a=[1,2,3,"hello",2]
a.extend([1,2,3])#追加一个数据容器(将里面的元素分别加入到a列表)
#a=[1,2,3,"hello",2,1,2,3]
a.extend("world")
#a=[1,2,3,"hello",2,1,2,3,'w','o','r','l','d']
```

## .strip()

**功能：删除开头与结尾指定元素，不改变原字符串，返回一个新的字符串**

如果不传参数，默认删除字符串开头和末尾的空格和控制字符（\n \t）

传入参数的话就删除指定元素，**注意，只能删除开头和结尾的元素**，不能指定删除中间的元素

## .split()

**功能：以指定元素分割字符串，返回为列表**

参数：指定的元素，如果不传参，默认以空格分隔

```python
a = "abc d e"
b = a.strip()
print(b) ---['abc','d','e']
```

## .replace()

**功能:将指定元素替换为指定元素，返回新的字符串**

参数：1.要替换的元素 2.替换为的元素



**字符串不能直接修改元素，如str[0] = 'k'**

## .difference()/.difference_update
